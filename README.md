# ☁ SkyCast Pro — Weather Forecast Web App

SkyCast Pro is a modern multi-page weather forecasting web application built using HTML, CSS, and JavaScript.
It provides real-time weather updates, hourly forecasts, favorite cities management, user authentication, responsive design, and light/dark theme support.

## ✨ Features

* 🌤 Real-time weather data
* 📍 Search weather by city
* 🛰 GPS location weather support
* ⏰ 24-hour forecast
* 📅 5-day weather forecast
* ⭐ Save favorite cities
* 🔐 User login & registration system
* 🌙 Light / Dark mode
* 📱 Fully responsive UI
* 📊 Weather charts & insights
* 💬 Feedback page
* 🖼 Interactive weather gallery

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* WeatherAPI.com API
* Font Awesome
* Chart.js

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/IqraBatoolAbbas/SkyCast-Pro.git
```

### 2. Open Project

Open the project folder in VS Code.

### 3. Add API Key

Create or open `config.js` and add your WeatherAPI key:

```javascript
const API_KEY = "YOUR_API_KEY";
```

Get free API key from:

[https://www.weatherapi.com/](https://www.weatherapi.com/)

### 4. Run Project

Open `index.html` in browser
or use VS Code Live Server.
